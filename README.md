# Bela
My experiments with my DIY DSP-box based on the Bela GEM platform

![My box](http://www.jeroenvanzutphen.nl/projects/more/bela.jpg "Bela GEM")

I've built a box with 8 analog potentiometers and 4 blue leds. All connected to a Bela GEM board. It has a stereo input and output. A 9v power port (converted inside to 5v). I use it to try out various DSP algorithms. This Github documents those algorithms, and makes them available to others.

I've had help from Claude AI with these files.
